# Day 6 - 字典 + JSON 永久存储

**日期**：2025年11月22日  
**最大成就**：程序关掉再打开，数据竟然还在！！！

## 今日核心收获

1. 字典才是最适合“姓名→成绩”这种键值对数据的结构
2. `json.dump()` + `json.load()` = Python 数据持久化的万能钥匙
3. `with open(...)` 自动关闭文件，再也不用记 f.close()
4. 每次修改后立刻 `save_data()`，永不丢数据
5. `os.path.exists()` 判断文件是否存在，程序第一次运行也不会报错

**从今天起，所有的项目都必须带永久存储！**
